# 🚀 Система автоматической персонализированной Email-рассылки (Python + n8n + Telegram)

Проект представляет собой автоматизированную систему для массовой персонализированной рассылки писем по базе клиентов из Excel-таблицы с мгновенными уведомлениями в Telegram-бота о статусе доставки.

## 🛠 Технологический стек
* **Python** (GUI-интерфейс, отправка вебхуков, обработка файлов)
* **n8n** (Node-based платформа для автоматизации бизнес-процессов)
* **Gmail SMTP** (Сервер отправки электронной почты)
* **Telegram Bot API** (Система логирования и уведомлений в реальном времени для администратора)
* **JavaScript** (Написание логики кастомизации шаблонов внутри n8n)

---

## 📐 Архитектура и схема работы системы

Процесс полностью автоматизирован и разбит на следующие этапы:

1. **Запуск из Python (`app.py`)**: Пользователь загружает Excel-файл с контактами через интерфейс приложения, указывает тему и HTML-шаблон письма, после чего жмет «Запустить». Python собирает данные и отправляет их единым пакетом на `Webhook` в n8n.
2. **Парсинг и итерация**: n8n принимает пакет данных, а узел `Item Lists` разбивает массив строк из таблицы на отдельные элементы (по одному на каждого клиента).
3. **Персонализация (JS-логика в `Code`)**: В узле `Code` для каждого клиента динамически генерируется контент: теги `{{name}}`, `{{company}}` и `{{email}}` в HTML-шаблоне автоматически заменяются на реальные данные текущего человека. При этом n8n сохраняет контекст для последующих шагов.
4. **Отправка почты (`Send Email`)**: Узел `Send Email` авторизуется через Gmail SMTP и отправляет готовое персонализированное письмо получателю.
5. **Логирование в Telegram**: После успешного ответа от почтового сервера узел `Telegram` обращается к Telegram Bot API, забирает исторические данные из узла `Code` и присылает администратору отчет в формате: 
   * *"🤖 Хозяин, я успешно отправил email-письмо для [Имя] на почту [Email]!"*

## 🚀 Как запустить проект самостоятельно

### 1. Настройка n8n
1. Создайте новый сценарий (Workflow) в вашем аккаунте n8n.
2. Настройте цепочку узлов: `Webhook` ➡️ `Item Lists` ➡️ `Code` ➡️ `Send Email` ➡️ `Telegram`.
3. В узле **Send Email** настройте подключение к вашему SMTP Gmail (потребуется сгенерировать *Пароль приложения* в настройках безопасности аккаунта Google).
4. Создайте Telegram-бота через официального бота `@BotFather`, скопируйте токен в настройки узла Telegram. Укажите ваш личный `Chat ID` (его можно узнать через `@userinfobot`).
5. Чтобы бот корректно выводил имя и email, используйте в поле текста выражение:
   `🤖 Хозяин, я успешно отправил email-письмо для {{ $('Code').item.json.recipient_name }} на почту {{ $('Code').item.json.recipient_email }}!`
6. Активируйте сценарий (нажмите **Publish**). Скопируйте URL созданного вебхука (Production URL).
